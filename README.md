# 🛒 J-Tech Web (Pré-Projeto de Loja Online)

Este projeto representa a fase inicial de desenvolvimento de uma loja virtual chamada **J-Tech**, criada com foco em tecnologia e vendas online.

## 🚀 Sobre o projeto
A J-Tech Web é um **pré-projeto de e-commerce**, desenvolvido para estruturar a base visual e funcional de uma loja online.

O objetivo é evoluir este projeto para uma aplicação completa de vendas.

## 🛍️ Funcionalidades atuais
- Estrutura inicial de um site de loja
- Layout organizado para exibição de produtos
- Navegação básica entre páginas
- Design focado em e-commerce

## 🛠️ Tecnologias utilizadas
- HTML5
- CSS3

## ▶️ Como executar
Basta baixar os arquivos ou abrir o `index.html` no navegador.
